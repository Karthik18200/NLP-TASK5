 NLP Assignment 5 – Fine-Tuning BERT for POS Tagging & Chunking

This project is part of my Data Science Internship (February 2026).
The objective is to build a token classification system using transformer models like BERT / DistilBERT to perform:

 Part-of-Speech (POS) Tagging
 Chunking (Phrase Detection)

The project demonstrates how modern NLP models can understand sentence structure and grammatical relationships.

Objectives
Perform token classification using transformer models
Implement POS tagging and chunking tasks
Handle tokenization & label alignment challenges
Evaluate models using sequence-based metrics

Technologies Used
Python 
Hugging Face Transformers 
PyTorch / TensorFlow
